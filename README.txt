Learn about the original game at https://tellstones.com/en-us/

How to run:

1. First run the server. The command line should look like:
	python3 server.py -p <port number>

2. Run the first client. The command line should look like: 
	python3 client.py -s <server ip address> -p <server port address>
	
3. Run the second client. The command line should look the same as the first client.
